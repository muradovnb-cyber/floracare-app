# 🌿 FloraCare — Универсальное приложение

Уход за растениями по подписке · Ташкент

## Что внутри
- 📱 Клиентское приложение (онбординг, кабинет, отчёты, оплата)
- 🔧 Приложение сотрудников (садовник + администратор)
- 🌐 Сайт-лендинг (премиальный тёмный дизайн)

## Аккаунты для демо

### Клиент
Войти через экран выбора роли → «Я клиент»

### Сотрудники
- Садовник: `maria` / `1234`

### Администраторы

- Admin 1 (ограниченный): `admin1` / `admin1`

## Деплой на Railway

1. Загрузить на GitHub
2. Подключить репозиторий в Railway
3. Railway автоматически запустит `npm start`

## Локальный запуск
```bash
npm install
npm start
# Открыть http://localhost:3000
```
